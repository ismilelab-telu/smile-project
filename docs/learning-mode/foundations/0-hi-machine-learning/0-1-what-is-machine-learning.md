# 0.1: Apa Itu Machine Learning

Status: Draft  
Path: Machine Learning Foundations  
Module: 0. Hi, Machine Learning!

## Goal

Membuat learner memahami machine learning sebagai cara komputer belajar pola dari contoh, bukan sekadar kumpulan aturan manual.

## Material

Machine learning adalah pendekatan dalam kecerdasan buatan yang membuat komputer dapat belajar dari data.

Arthur Samuel memperkenalkan istilah machine learning pada tahun 1959 dan mendefinisikannya sebagai:

"A field of study that gives computers the ability to learn without being explicitly programmed."

Artinya, komputer tidak selalu diberi aturan satu per satu untuk setiap kondisi. Dalam machine learning, kita memberi contoh, data, dan tujuan. Sistem lalu mencari pola dari contoh tersebut.

Contoh sederhana:

- Program aturan manual: "Jika suhu di atas 30 derajat, rekomendasikan minuman dingin."
- Machine learning: sistem melihat banyak riwayat suhu, waktu, promosi, dan penjualan, lalu belajar pola permintaan dari data tersebut.

Machine learning berguna ketika pola terlalu banyak, berubah-ubah, atau sulit ditulis sebagai aturan manual yang lengkap.

Machine learning bukan berarti komputer memahami dunia seperti manusia. Model hanya belajar dari data, tujuan, dan feedback yang kita berikan.

## Exercise

Type: single-choice

Prompt: Pernyataan mana yang paling tepat menjelaskan machine learning?

Options:

- A. Komputer menjalankan daftar aturan tetap yang ditulis manusia untuk semua kondisi.
- B. Komputer belajar pola dari data agar dapat membuat prediksi, rekomendasi, atau keputusan untuk contoh baru.
- C. Komputer selalu menghasilkan jawaban benar karena sudah memakai kecerdasan buatan.
- D. Komputer tidak membutuhkan data karena model sudah mengetahui pola dari awal.

Answer Key:

- Correct: B

## Feedback

Jawaban yang tepat menekankan bahwa machine learning belajar dari data. Aturan manual tetap bisa berguna, tetapi machine learning dipakai ketika pola perlu dipelajari dari contoh.

## Hints

1. Cari jawaban yang menyebut belajar dari data.
2. Machine learning tidak menjamin jawaban selalu benar.
3. Model membutuhkan contoh sebelum bisa belajar pola.
