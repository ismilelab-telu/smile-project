Status: Draft  
Path: Machine Learning Foundations

## Goal

Explain loading data into a workable table or dataframe.

## Material

Data loading turns raw sources into a table the project can inspect.

After loading, check the shape, column names, data types, and a few preview rows.

Loading is not modeling yet. It is the step where we make sure the data arrived correctly.
