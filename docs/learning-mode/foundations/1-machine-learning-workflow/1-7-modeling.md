# 1.7: Modeling

Status: Draft  
Path: Machine Learning Foundations  
Module: 1. Machine Learning Workflow

## Goal

Place modeling as one step inside the wider ML workflow.

## Material

Modeling is where an algorithm learns from training data.

It should not be the first step. The data, target, features, split, and evaluation idea must be clear first.

At the foundations level, learners only need to understand where modeling fits.

## Exercise

Choose actions that belong in modeling.

## Feedback

Explain that a model should be compared with a simple baseline and evaluated after training.

## Hints

1. Modeling learns from training data.
2. A baseline gives a simple comparison point.
3. Evaluation checks whether the model helps.
