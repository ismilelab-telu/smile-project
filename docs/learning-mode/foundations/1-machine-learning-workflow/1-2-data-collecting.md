# 1.2: Data Collecting

Status: Draft  
Path: Machine Learning Foundations  
Module: 1. Machine Learning Workflow

## Goal

Explain data collection as a purposeful step, not random column hoarding.

## Material

Data collecting starts from a question.

The team needs to know what problem the model should help solve, where the data comes from, and whether the data can be used safely.

Useful data collection records source, timing, permissions, and context.

## Exercise

Choose responsible data collection habits.

## Feedback

Push against collecting everything without a prediction goal.

## Hints

1. Start from the question.
2. Record the source.
3. Check permission and relevance.
