# 1.4: Data Cleaning and Transformation

Status: Draft  
Path: Machine Learning Foundations  
Module: 1. Machine Learning Workflow

## Goal

Introduce cleaning and transformation as preparation steps before analysis or modeling.

## Material

Cleaning handles problems such as missing values, inconsistent labels, duplicates, and impossible values.

Transformation reshapes data into a more useful form.

Both steps need care because careless changes can hide problems or leak the answer into the inputs.

## Exercise

Choose which actions belong in cleaning and transformation.

## Feedback

Separate useful preparation from risky shortcuts.

## Hints

1. Cleaning fixes trust problems.
2. Transformation changes representation.
3. Do not use future answer information as input.
