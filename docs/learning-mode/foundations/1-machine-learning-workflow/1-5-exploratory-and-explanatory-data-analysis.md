# 1.5: Exploratory and Explanatory Data Analysis

Status: Draft  
Path: Machine Learning Foundations  
Module: 1. Machine Learning Workflow

## Goal

Separate exploration from explanation.

## Material

Exploratory data analysis is for looking around: distributions, relationships, gaps, and surprises.

Explanatory analysis is for communicating a finding clearly.

A chart can support a claim, but the claim must not go beyond the evidence.

## Exercise

Choose statements that describe exploratory and explanatory analysis.

## Feedback

Emphasize evidence and avoid overclaiming.

## Hints

1. Exploration asks questions.
2. Explanation communicates selected findings.
3. A chart is evidence, not automatic proof.
