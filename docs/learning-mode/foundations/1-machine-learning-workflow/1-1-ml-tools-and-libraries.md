# 1.1: ML Tools and Libraries

Status: Draft  
Path: Machine Learning Foundations  
Module: 1. Machine Learning Workflow

## Goal

Introduce the tool roles used in a basic machine learning workflow.

## Material

Machine learning work usually uses a few kinds of tools.

A notebook or code editor is where experiments are written and run.

A dataframe library helps load, inspect, clean, and transform tabular data.

A modeling library provides algorithms, metrics, and utilities for training and evaluation.

## Exercise

Choose which tools belong in a practical ML workflow.

## Feedback

Explain that tools support the workflow but do not replace reasoning.

## Hints

1. Experiments need an environment.
2. Tables need inspection and transformation tools.
3. Models need training and evaluation utilities.
