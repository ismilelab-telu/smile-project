# Smile Project

Alias: Bun Playground

Project kecil berbasis Bun untuk eksperimen JavaScript/TypeScript.

## Struktur

- `data/raw/` untuk dataset mentah.
- `data/processed/` untuk dataset hasil olahan.
- `src/` untuk kode TypeScript.
- `models/` untuk model atau artifact hasil training.
- `tests/` untuk pengecekan dengan `bun test`.

## Mulai Cepat

```bash
cd ~/Documents/"Smile Project"
bun install
bun run dev
```

Untuk menjalankan test:

```bash
bun test
```
