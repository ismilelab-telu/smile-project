Machine learning is an approach inside artificial intelligence that lets computers learn from data. The main point is not making a computer "smart" by magic, but building a system that can find patterns from examples we provide.

Arthur Samuel introduced the term machine learning in 1959 and described it as:

> "A field of study that gives computers the ability to learn without being explicitly programmed."

This means we do not always write one rule at a time for every possible condition. We provide data, examples, and a goal. From there, the system tries to adjust its patterns so it can produce predictions, recommendations, scores, or decisions for new examples.

## From Rules to Patterns

Manual rule-based programs work when the business rules are clear and stable. For example, "if the distance is under 2 km, estimate delivery at 15 minutes." Rules like this are easy to understand, but they quickly become complicated as the number of conditions grows.

Machine learning takes a different approach. The system looks at many records of distance, weather, traffic level, time of day, vehicle type, preparation time, courier experience, and delivery time. The model then learns relationships that often appear in the data, such as longer routes with high traffic tending to need longer delivery times.

## When This Is Useful

Machine learning makes sense when:

- there are too many patterns to write manually;
- patterns change over time;
- decisions need to be made repeatedly for many examples;
- relevant data is available to learn from;
- the result can be evaluated clearly.

Machine learning still has limits. A model does not understand the world like a human. It only learns from the data, target, and feedback that are available. If the data is poor, the target is wrong, or the context changes, the model can also produce poor answers.

So the core idea of machine learning is learning patterns from data, not running a fixed list of rules for every condition.
