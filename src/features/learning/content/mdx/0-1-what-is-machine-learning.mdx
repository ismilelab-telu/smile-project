Machine learning adalah pendekatan dalam kecerdasan buatan yang membuat komputer belajar dari data. Fokus utamanya bukan membuat komputer "pintar" secara ajaib, tetapi membuat sistem mampu menemukan pola dari contoh yang kita berikan.

Arthur Samuel memperkenalkan istilah machine learning pada tahun 1959 dan menjelaskannya sebagai:

> "A field of study that gives computers the ability to learn without being explicitly programmed."

Artinya, kita tidak selalu menulis aturan satu per satu untuk setiap kondisi. Kita memberi data, contoh, dan tujuan. Dari sana, sistem mencoba menyesuaikan pola agar bisa menghasilkan prediksi, rekomendasi, skor, atau keputusan untuk contoh baru.

## Dari Aturan ke Pola

Program aturan manual bekerja ketika aturan bisnisnya jelas dan stabil. Misalnya, "jika suhu di atas 30 derajat, rekomendasikan minuman dingin." Aturan seperti ini mudah dipahami, tetapi cepat menjadi rumit ketika kondisi makin banyak.

Machine learning mengambil pendekatan berbeda. Sistem melihat banyak riwayat suhu, waktu shift, promo, cuaca, dan penjualan. Model lalu belajar hubungan yang sering muncul di data, misalnya shift sore saat cuaca panas dan promo aktif cenderung punya permintaan lebih tinggi.

## Kapan Ini Berguna

Machine learning masuk akal ketika:

- pola terlalu banyak untuk ditulis manual;
- pola berubah dari waktu ke waktu;
- keputusan harus dibuat berulang pada banyak contoh;
- tersedia data yang relevan untuk dipelajari;
- hasilnya bisa dievaluasi dengan cara yang jelas.

Machine learning tetap punya batas. Model tidak memahami dunia seperti manusia. Model hanya belajar dari data, target, dan feedback yang tersedia. Kalau data buruk, target keliru, atau konteksnya berubah, model juga bisa menghasilkan jawaban yang buruk.

Jadi, inti machine learning adalah belajar pola dari data, bukan menjalankan daftar aturan tetap untuk semua kondisi.
